I have been building a small "os" for the command prompts of Windows based computers. It is written in Java. It uses an
interface file to allow for multiple class files to be created as `Modules` that can be "imported" into the os
before launch.
<br>
See the below table for info on which features are still in development.

|Feature|Implemented?|
|-------|------------|
|Basic Functionality|yes|
|Module Framework|yes|
|Working module|yes|
|Advanced modules|no|
|Documentation|[yes](https://github.com/geekkid1/IsaacOS)|
